# Rust REST API 

REST API built with Rust lang. It uses [Rocket](https://rocket.rs/) HTTP framework.

### Libraries used
- fraction = "0.11.1"
- rand = "0.8.5"
- rocket = "0.4.11"
- rocket_contrib = { version = "0.4.6", features = ["json"] }

### Use

#### Build Dependencies
```sh
cargo build
```

### Run App
```sh
cargo run
```
